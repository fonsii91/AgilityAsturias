🐕 Si estás en Grado 2 (Para pasar a Grado 3):

Las reglas se endurecen y ya no hay vías opcionales; debes completar ambos retos bajo la mirada de 3 jueces diferentes en cada disciplina:

  • military_tech Agility: 3 pistas a 0 cumpliendo una velocidad mínima (4.50 m/s para S/M y 4.70 m/s para L/X).
  • military_tech Jumping: 3 pistas a 0 cumpliendo una velocidad mínima (4.70 m/s para S/M y 4.90 m/s para L/X).
