help_outline Instrucciones: Material y Recursos

people 1. Reglas para los Miembros

En esta sección encontrarás todo el material formativo y de interés. Haz clic en cualquier recuadro para abrir su contenido. Los iconos de cada tipo de contenido son:

  • play_circle
    Vídeos: Para visualizar contenido en vídeo.
  • link
    Enlaces: Para redirigirte a otras páginas web de interés.
  • description
    Documentos: Para abrir y leer archivos como PDFs.
  • filter_list
    Filtrado: En la parte superior dispones de filtros para acotar los recursos por Categoría (ej. Dietas) y por Nivel (ej. Iniciación).
  • warning
    ⚠️ Aviso de filtros: ¡Ojo con mezclar filtros! Si seleccionas a la vez una Categoría y un Nivel concreto, el sistema buscará cosas que cumplan ambas condiciones. Si la lista se queda vacía, es posible que no haya nada con esa combinación exacta. Puedes seleccionar "Todas" de nuevo para recuperarlos.
  • refresh
    Refrescar: Si crees que hay material nuevo que no aparece, puedes usar el botón de Recargar en cualquier momento para actualizar la lista.

admin_panel_settings 2. Opciones de Staff

Como personal del equipo tienes acceso a opciones exclusivas de gestión:

  • add_circle_outline
    Añadir Recurso: Este botón superior permite subir archivos nuevos o compartir enlaces, y asociarles el nivel mínimo y categoría correspondientes.
  • edit delete
    Gestión de contenido: Dentro de cada recurso (al pasar el ratón por encima en ordenador o al mantener presionado con el móvil), aparecerán los botones para editar el material subido o borrarlo permanentemente.
